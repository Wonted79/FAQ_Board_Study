# 📚 FAQ Board System

Spring Boot 기반 FAQ 게시판 시스템입니다.  
고객사 코드 기반 회원가입 및 세션 로그인 기능을 포함합니다.

---

## 🚀 Tech Stack

- Java 21+
- Spring Boot
- Thymeleaf
- MariaDB
- MyBatis / JPA
- HTML / CSS
- Git

---

## 📌 주요 기능

### 🔐 인증 / 인가
- 세션 기반 로그인
- 이메일 인증 회원가입 
- 다양한 고객사별 기능 분리를 고려하여 고객코드 사용 
- 
### 📄 FAQ 기능
- FAQ 목록 조회
- 카테고리별 조회
- 검색 기능
- FAQ 상세 조회

### 🏢 관리자 기능 (예정)
- FAQ 등록 / 수정 / 삭제
- 카테고리 관리
- 1:1 문의 관리
